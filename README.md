# Rubiks Cube Simulator and Solver

## Controls

- Use the arrow keys or drag with the left mouse button to orbit the cube.
- Use the mouse wheel to zoom in or out.
- Press `U`, `D`, `F`, `B`, `L`, or `R` to animate a clockwise quarter-turn of that face.
- Hold `Shift` while pressing a face key to animate the counter-clockwise turn.
- Press `Esc` to exit.
